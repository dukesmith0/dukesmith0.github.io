Selected work

6-DOF UAV Flight Simulator

MATLAB Flight Dynamics Quaternion EOM PID Autopilot Monte Carlo

6-DOF rigid body flight dynamics simulator in MATLAB with quaternion-based equations of motion and stability-derivative aero models. Trim validated within 1% of hand calculations. 7-loop cascaded PID autopilot achieving <0.01 m altitude error and 0.03° heading error. All 5 classical flight modes identified via eigenvalue analysis. Monte Carlo wind-gust sensitivity analysis. 20 formal unit tests and energy/momentum conservation checks confirming <0.1% drift.

GitHub

PyTherm

Python NumPy Heat Transfer FDM PyQt6

2D transient finite-difference thermal solver in Python achieving 2000x speedup over global CFL stability bound via per-cell adaptive time-stepping with harmonic-mean interface conductances. Aerospace TPS scenarios (aerogel/alumina/titanium under 200 W/m² heat flux) with Dirichlet and Neumann boundary conditions. Validated with per-frame energy conservation across 196 Incropera materials.

GitHub

Flight Data Recorder + Visualizer

RP2040 Embedded C++ IMU/GPS Mahony AHRS Streamlit

Dual-core RP2040 data acquisition system for FSAE CFD validation, integrating IMU, GPS, and barometric instrumentation with Mahony AHRS sensor fusion at 200 Hz and $113 total BOM. Custom signal processing pipeline with GPS quality rejection, complementary altitude filtering, and automated calibration. Web-based 3D trajectory visualizer with dead reckoning for GPS-denied intervals; adopted by a collaborator for AI-based activity classification.
