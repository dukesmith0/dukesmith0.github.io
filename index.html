Selected work

6-DOF UAV Flight Simulator

MATLAB Simulink Flight Dynamics Quaternion EOM PID Autopilot Monte Carlo

Achieved <0.01 m altitude error and 0.03° heading error with a 7-loop cascaded PID autopilot on a full 6-DOF rigid-body flight simulator. Quaternion-based EOM avoid gimbal lock at high alpha; stability derivatives scaled from Nelson textbook data with RK4 integration. Trim validated within 1% of hand calculations. All 5 classical flight modes (short period, phugoid, dutch roll, roll subsidence, spiral) identified via eigenvalue analysis of the linearized plant. Autopilot robustness characterized through 100-run Monte Carlo gust sensitivity across 3 Dryden turbulence levels. Verified with 20 formal unit tests and energy/momentum conservation checks confirming <0.1% drift.

PyTherm

Python NumPy Heat Transfer FDM PyQt6

2,000x speedup over global CFL stability bound on mixed-material grids via per-cell adaptive time-stepping with harmonic-mean interface conductances. Enables real-time parameter sweeps that were previously overnight batch jobs. Modeled aerospace TPS scenarios (aerogel/alumina/titanium under 200 W/m² heat flux) with Dirichlet and Neumann boundary conditions. Validated with per-frame energy conservation across 196 Incropera materials and grid independence verified across multiple resolutions with <1% deviation. Shipped 6 versioned releases with a 101-test regression suite and cross-platform executables.

Flight Test Instrumentation Package

RP2040 Embedded C++ Sensor Fusion Mahony AHRS Streamlit

Mahony AHRS attitude estimator fusing 6-axis IMU data at 200 Hz on a dual-core RP2040, selected over extended Kalman filter due to Cortex-M0+ compute constraints at target update rate. GPS-disciplined timestamps with quality-based fix rejection for data conditioning. Complementary altitude filtering and automated magnetometer/accelerometer calibration. Dead-reckoning navigation propagates position through GPS-denied intervals. $113 BOM flight test instrumentation package built for FSAE CFD validation with 100 Hz fault-tolerant SD logging. Web-based 3D trajectory visualizer adopted by a collaborator for AI-based activity classification.
